# Técnico em Desenvolvimento de Sistemas (TDS)

Este repositório reúne as atividades, exercícios, projetos práticos e materiais de apoio desenvolvidos pelas turmas do curso Técnico em Desenvolvimento de Sistemas do Senac Montenegro.

Aqui você encontrará:

- Projetos individuais e em grupo  
- Práticas de programação  
- Exercícios guiados  
- Materiais complementares  
- Documentações e orientações

Este espaço foi criado para incentivar a colaboração, a organização e o crescimento dos alunos na área de tecnologia.
